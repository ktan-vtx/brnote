# brnote（ブラノート） - ブラウザで1秒で開く、極上デザインのシンプルメモ帳

`brnote` は、面倒なソフトのインストールや会員登録が一切いらない、ブラウザだけで動く高機能で美しいメモ帳アプリです。

ブックマーク（お気に入り）に登録しておくだけで、いつでも瞬時に起動し、書いた内容はすべてあなたのパソコンの中に自動で保存されます。

---

以下のリンクをクリックして開き、ブラウザのブックマーク（お気に入り）に追加してください。

## ✨ `brnote` が選ばれる5つの理由

### 1. 圧倒的な安心感 — 外部に漏れない完全プライベート
あなたが書いたメモや貼り付けた画像は、**すべてあなた自身のパソコン（ブラウザの内部ストレージ）にだけ保存されます。**
外部のサーバーやインターネット上へ送信されることは一切ないため、仕事のメモ、プライベートな日記、パスワードの一時控えなども安心して書き留めることができます。

### 2. 「保存し忘れた！」がない完全自動保存
文字を入力するたびに、一瞬で自動保存されます。タブをうっかり閉じたり、パソコンが突然再起動したりしても、再び同じURLを開けば、さっきまで書いていた内容がそのまま復元されます。

### 3. 写真（画像）の貼り付け・ドラッグ自由配置 ＆ ファイル追加
コピーした画像を貼り付けたり、パソコンから画像ファイルを直接ドラッグ＆ドロップして追加できます。
*   **貼り付け & ドロップで追加**: `Ctrl` + `V` での貼り付けに加え、**パソコン内の画像ファイルを直接エディタにドラッグ＆ドロップするだけ**で、狙った位置に瞬時に画像を追加できます。
*   **サイズ変更**: 画像の右下をマウスでドラッグして、自由に大きさを変えられます。
*   **ドラッグ移動**: 貼り付けた画像全体をドラッグ＆ドロップして、文字の隙間など好きな位置へ自在に移動できます。
*   **テキストの回り込み**: 画像の上をホバー（マウスを載せる）すると表示されるボタン（◀ / ■ / ▶）を押すことで、画像の左右に文章を回り込ませて配置できます。
*   **画像専用メモ**: 貼り付けた画像の下には、その画像専用のミニコメント欄が自動で作成されます。

### 4. 気分に合わせて選べる13種類の美しいデザイン（テーマ）
上質なすりガラス調（グラスモフィズム）のツールバーを採用。枠線をなくした極上のモダンデザインで、以下の13種類のテーマからワンクリックで切り替えられます。
*   **Carbon（黒曜石）**: 落ち着いた定番のダークテーマ
*   **Chalk（白磁）**: 清潔感のある明るいライトテーマ
*   **Sepia（羊皮紙）**: 目に優しいアンティーク調
*   **Ocean（深海）** / **Forest（常盤）**: 深みのあるブルーとグリーン
*   **Sakura（桜色）**: 華やかで可愛らしい淡いピンク
*   **Nord（北欧）**: 冷涼でスタイリッシュなブルーグレー
*   **Midnight（漆黒）**: OLEDディスプレイに最適な完全な黒
*   **Cyberpunk（未来）**: 鮮やかなネオンカラー
*   **Dracula（吸血鬼）**: 定番のおしゃれな暗色
*   **Sunset（夕暮れ）**: 暖かい夕暮れのグラデーション
*   **Wasi（和紙）**: 伝統的な日本の紙のぬくもり
*   **Matrix（仮想世界）**: レトロなSF風の緑と黒

### 5. 音声で書く（音声入力）
マイクアイコンをクリックするか、キーボードの `Alt` + `V` を押すだけで、話した言葉がそのままリアルタイムで文字に変換され、入力されます。アイデアのメモや日記の口述筆記に最適です。

## 🛠️ その他の便利な機能

*   **自動で隠れるツールバー**: 執筆に集中できるよう、文字を入力している間は上部のツールバーが自動的に隠れます。画面最上部にマウスを持っていくだけでいつでも再表示できます。
*   **文字数・単語数カウンター**: 画面右下に、現在のリアルタイムな文字数と単語数が表示されます。
*   **複数の形式で保存（書き出し）**:
    *   **全コピー**: 1クリックで書いた内容をすべてコピーします。
    *   **テキスト (.txt) 保存**: プレーンなテキストファイルとして保存します。
    *   **Markdown (.md) 保存**: 画像やコメント、太字などのレイアウトを維持したまま、他のメモアプリで開ける形式で保存します。
    *   **自分だけのアプリ (.html) 保存**: 今書いたメモの内容をそのまま埋め込んだ、あなた専用の新しい `brnote` ファイルとして丸ごとダウンロードします。

## ⌨️ 便利なキーボード操作（ショートカット）
*   **音声入力を始める / 止める**: `Alt` + `V`
*   **文字を大きくする**: `Alt` + `+`（または `=`）
*   **文字を小さくする**: `Alt` + `-`
*   **選んだ文字を太字にする**: `Ctrl` + `B`
