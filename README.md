# brnote | 超軽量・高機能ミニマルメモ帳

`brnote` は、サーバー（ホスティング）不要で、PCのローカルファイル（HTML）を直接ブラウザで開くだけで1秒で起動・動作する、完全自己完結型の極小メモ帳アプリです。

---

## 🚀 サーバー不要！ローカルファイルとして動かす方法
本アプリは**ローカルホストを立ち上げる必要はありません**。
1. エクスプローラーなどで [index.html](./index.html) を探します。
2. そのファイルをダブルクリックしてブラウザで直接開きます（URL欄が `file:///...` から始まる状態になります）。
3. あとは、そのページをブラウザの**ブックマークバーに登録**しておくだけで、いつでも瞬時にブラウザの JavaScript とデータベース（IndexedDB）だけでメモが動き、自動保存されます。

---

## 🌟 主な機能と特徴

### 1. ボーダーレス・グラスモフィズムUI & オートハイド
- **最上級のグラスUI**: ツールバーやドロップダウン、モーダル、トースト、画像カードに至るまで、枠線（ボーダー）を一切排除したシームレスなデザインを採用。20〜24pxの強力な背景ぼかしとインセットハイライト（内側シャドウ）によって、極上のガラスのような高級質感を演出します。
- **オートハイド**: 画面右上にある丸い「上矢印 (▲)」ボタンを押すと、執筆中はツールバーが自動的に隠れます。カーソルを画面最上部に近づけるか、丸い「下矢印 (▼)」ボタンを押すだけでいつでも再表示できます。
- **全画面表示 (常時)**: 画面全体（画面幅96%）を最大限に広く使った執筆領域を常に確保しています。

### 2. タイポグラフィと行間の微調整
- **部分的な太字化**: 太字にしたいテキスト範囲を選択した状態でツールバーの **太字アイコン** または `Ctrl` + `B` を押すと、**選択部分だけが部分的に太字になります**。
- **フォントの種類 (Aa アイコン)**: Sans-Serif (モダン)、Serif (明朝・小説風)、Monospace (プログラミング風) の切り替え。
- **行間の微調整**: ツールバーの **行間アイコン** ドロップダウンから、`1.0`（狭め）〜 `2.5`（広め）の5段階の行間を選択できます。
- **多彩なテーマ (13種類)**: ツールバーのパレットアイコンから、Carbon(黒曜石)、Chalk(白磁)、Sepia(羊皮紙)、Ocean(深海)、Forest(常盤)、Sakura(桜)、Nord(北欧)、Midnight(漆黒)、Cyberpunk(サイバー)、Dracula(吸血鬼)、Sunset(夕暮れ)、Wasi(和紙)、Matrix(仮想世界) の全13種類のハイクオリティテーマを切り替えられます。

### 3. 画像の貼り付け・自在なレイアウト配置 ＆ サイズドラッグ調整
- **即時貼り付け**: 画像をコピーし、エディタ上で `Ctrl` + `V` を押すだけでその場に画像を即時貼り付けられます。
- **ドラッグ＆ドロップ移動**: 貼り付けた画像ブロック全体を**マウスでドラッグし、エディタ内の任意の行やテキストの間にドロップして自由に移動**（ペタンと再配置）させることができます。
- **テキストの回り込み（レイアウト切り替え）**: 画像ブロックをホバーすると上部に表示されるアライメントボタンから配置を切り替えられます：
  - `◀` (左寄せ): 画像を左に寄せ、テキストを右側に美しく回り込ませて配置します。
  - `■` (中央寄せ): テキストの回り込みを解除し、標準的な中央表示にします。
  - `▶` (右寄せ): 画像を右に寄せ、テキストを左側に美しく回り込ませて配置します。
- **サイズ調整**: 画像をホバーすると右下に表示されるハンドルをドラッグして、サイズを無段階に調整できます。
- **コメント欄**: 画像の下に専用のミニコメント欄が自動生成され、画像を補足するメモを残せます。

### 4. カラム分割（2列レイアウト）
- ツールバーの `[||]` アイコンから左右分割カラムを挿入できます。カラム間の境界線（セパレーター）を左右にドラッグすることで、列の幅比率を自由に変更できます。

### 5. クイックコピー ＆ 汎用的なエクスポート
他のノートアプリやファイルと柔軟に連携できるよう、出力を複数の形式で行うことができます。
- **全コピーボタン**: ツールバーのコピーアイコンをクリックするだけで、エディタの全テキストを一瞬でクリップボードにコピーします。
- **Markdown (.md) で保存**: 画像やコメントカード、カラム、太字修飾などを維持したまま、他のアプリで開ける**汎用的な Markdown 形式**に変換してダウンロードします。
- **テキスト (.txt) で保存**: メモをプレーンなテキストファイルとして保存します。
- **自己完結 HTML (.html) で保存**: 現在書いている内容そのものを内部に焼き込んだ、新しい `brnote` HTMLファイルとして保存します。

## ⌨️ キーボードショートカット

- **フォントサイズ大きく**: `Alt` + `+`

## 🌐 GitHub Pagesでの無料ホスティングとメリット
`brnote` は完全に静的なHTMLファイルのため、GitHubのパブリック（公開）リポジトリに配置し、GitHub Pages機能を有効にするだけで**完全無料**でWeb上に公開・ホスティングできます。

### 💡 GitHub Pagesの無料枠・制限
- **費用**: 公開リポジトリからのPages利用は**完全無料**です。
- **データ転送量上限**: 毎月 100GB まで（個人メモ用途であればまず到達しません）。
- **リポジトリサイズ**: 推奨 1GB 未満（ハードリミット 10GB）。HTML1枚とファビコン画像のみのため数MB以下で十分収まります。
- **ビルド回数上限**: 1時間あたり10回まで。

### 🚀 Pages化する最大のメリット
1. **セキュアオリジン (HTTPS) によるAPI完全動作**:
   ブラウザのローカルファイル（`file://`）として直接開く場合、一部のブラウザのセキュリティ制限により「クリップボードコピー（Clipboard API）」や「音声入力（Web Speech API）」が制限される場合があります。HTTPSで配信されるGitHub Pages上に置くことで、これらのモダンなブラウザ機能が**制限なく100%正常に動作**します。
2. **データの永続化 (IndexedDB)**:
   IndexedDB はWebサイトのドメイン（オリジン）ごとに管理されます。GitHub PagesのURL（`https://ktan-vtx.github.io/brnote/`）にアクセスすれば、どのブラウザでも常に同じストレージが維持され、データが勝手に消えることなく強固に保存されます。
